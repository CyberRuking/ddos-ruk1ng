# ddos-ruk1ng

### Installation
```shell 
git clone https://github.com/CyberRuking/ddos-ruk1ng

cd ddos-ruk1ng

chmod +x ddosruk1ng.py

# Example
python3 ddosruk1ng.py -s target -t 300
```

### Usage
```shell
usage : python3 ddosruk1ng.py [-s] [-p] [-t]
-h : help
-s : server ip
-p : port default 80
-t : turbo default 135
```
